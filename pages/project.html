Voice Activated Systems

COMP 680 - Project

Business Goal:

Develop an Alexa skill that will offer customers a more intuitive way to interact with the CSUN University Calendar using voice commands. This will help the customers to know about upcoming events at CSUN by asking Alexa about events based on user's preferred time-frame. User can also ask Alexa for more detailed information about the event.

Sprint 1

Objective:

Creating a Basic Alexa Skill Set which will respond to a simple user request.

User Stories:

  1. Activate Alexa skill upon calling Invocation name
  2. Alexa will respond according to user voice request

Sprint 2

Objective:

Ec]xtend the Basic Alexa Skill Set which will respond to complex user requests.

User Stories:

  1. Ask Alexa for events based on time frame
  2. Ask Alexa for detailed info for an event
